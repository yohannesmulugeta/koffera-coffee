# Koffera Cinematic Image-Scroll Prompts

Generate the desktop images first in the order shown. Keep the same visual language, colour grade, realism level and Ethiopian context across all six scenes. Then generate each mobile image by using its approved desktop image as a visual reference and natively recomposing it for portrait orientation.

## Shared production rules

- Desktop: 16:9 landscape, ideally 1920×1080 or larger.
- Mobile: native 9:16 portrait, ideally 1080×1920 or larger; never crop the desktop image.
- Photorealistic cinematic documentary photography, not illustration or 3D.
- Premium international coffee-brand art direction.
- Warm espresso-brown, caramel, muted green and cream colour grade.
- Natural Ethiopian skin tones and realistic adult people.
- No fabricated Koffera facility, logo, certificate, customer, shipment number or ownership claim.
- No readable text, letters, labels, logos, flags, watermarks or product branding.
- No child labour, no stereotyped costume, no staged charity imagery.
- Realistic hands, faces, tools, coffee cherries, beans, bags and equipment.
