# Koffera Coffee Website Prototype

A responsive React + Vite prototype for rebuilding the Koffera Coffee Export website.

## Included

- Desktop-first and mobile-responsive layout
- Home, Coffees, About, Contact, Privacy, Terms and 404 routes
- Temporary Koffera product and brand assets sourced from the existing public website
- Buyer-oriented coffee catalogue placeholders
- Black media placeholders for future image/video links
- Structured editable content in `src/data/siteContent.js`
- Inquiry-form UI ready for backend connection
- Accessible navigation and responsive menu

## Important

This is a design and frontend prototype. Product specifications, certifications, company claims, legal identity, availability and contact details require verification before launch.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Media replacement

Replace files under `public/media/` or connect external video links inside `MediaPlaceholder` sections.
